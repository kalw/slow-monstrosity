What is this site?

This is a static website made by me as a test subject for improvement performance on the web and applying other best practices. The idea came after watching a couple of videos from JSConfEU 2014 by Addy Osmani and Patrick Hamann talking about performance improvement tooling and performance improvement best practices (mostly for CSS by Patrick Hamann).

After getting all excited about all the new tools and the best practices i wanted to test those things on a website. Even though the web is full of slow websites, having access to the full source code of it is not a simple thing. So what i did was create this site instead. Here i'm trying to make this site as slow as possible by applying as many anti-patterns as possible. The target is to get a very low score on Google pagespeed insights and in tools like yslow.
